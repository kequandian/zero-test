# Render PDF - mdpdf Skill

A professional Markdown to PDF conversion skill using **mdpdf** (Puppeteer-based rendering).

## Features

- **Full HTML/CSS Rendering** - Leverages Chromium's rendering engine
- **Custom CSS Styles** - Complete control over PDF appearance
- **Excellent Chinese Support** - Microsoft YaHei and other CJK fonts
- **GitHub-Style Formatting** - Professional, familiar look
- **Headers & Footers** - Configurable page headers and footers
- **Syntax Highlighting** - Code blocks with language detection

## Quick Start

### Installation

```bash
# Standard installation
npm install mdpdf

# China mirror (faster download)
set PUPPETEER_DOWNLOAD_HOST=https://mirrors.huaweicloud.com/chromium-browser-snapshots
npm install mdpdf
```

### Basic Usage

```bash
cd skills/render-pdf-mdpdf

# Convert markdown file to PDF
node scripts/render.js input.md output.pdf

# With custom CSS
node scripts/render.js input.md output.pdf --css assets/custom.css

# With header and footer
node scripts/render.js input.md output.pdf --header "Page %p" --footer "Generated by mdpdf"
```

### Programmatic Usage

```javascript
const { convertMarkdownToPDF, convertMarkdownStringToPDF } = require('./scripts/pdf-converter');

// Convert from file
await convertMarkdownToPDF('input.md', 'output.pdf');

// Convert from string
await convertMarkdownStringToPDF('# Hello\n\nWorld!', 'output.pdf');

// With custom CSS
await convertMarkdownToPDF('input.md', 'output.pdf', {
    css: 'body { font-family: "SimSun"; }'
});
```

## Customization

### Custom CSS

Create a custom CSS file:

```css
/* custom.css */
body {
    font-family: "SimSun", "Microsoft YaHei", serif;
    font-size: 12pt;
    line-height: 1.8;
}

h1, h2 {
    color: #2c3e50;
    border-bottom: 2px solid #3498db;
}

code {
    background-color: #f4f4f4;
    padding: 2px 6px;
    border-radius: 3px;
}

pre {
    background-color: #282c34;
    color: #abb2bf;
    padding: 16px;
    border-radius: 6px;
}
```

Use it:

```bash
node scripts/render.js input.md output.pdf --css custom.css
```

### Headers and Footers

```javascript
await convertMarkdownToPDF('input.md', 'output.pdf', {
    header: 'Document Title | Page %p',
    footer: 'Generated on %D | Confidential'
});
```

### Page Options

```javascript
await convertMarkdownToPDF('input.md', 'output.pdf', {
    mdpdfOptions: {
        pdf: {
            format: 'A4',
            orientation: 'portrait',
            border: {
                top: '1cm',
                right: '1cm',
                bottom: '1cm',
                left: '1cm'
            }
        }
    }
});
```

## Examples

### Example 1: Simple Conversion

```javascript
const { convertMarkdownToPDF } = require('./scripts/pdf-converter');

await convertMarkdownToPDF('README.md', 'README.pdf');
```

### Example 2: Custom Styling

```javascript
const { convertMarkdownStringToPDF } = require('./scripts/pdf-converter');

const markdown = `# Report

Generated on ${new Date().toLocaleString()}

## Summary

- Item 1
- Item 2
- Item 3
`;

await convertMarkdownStringToPDF(markdown, 'report.pdf', {
    css: `
        body { font-family: "Arial"; }
        h1 { color: #0066cc; }
        h2 { color: #0066cc; border-bottom: 1px solid #ccc; }
    `
});
```

### Example 3: Test Report Generation

```javascript
const { convertMarkdownStringToPDF } = require('./scripts/pdf-converter');

function generateTestReport(testResults) {
    const markdown = `# Test Report

## Summary

- Total Tests: ${testResults.total}
- Passed: ${testResults.passed}
- Failed: ${testResults.failed}
- Duration: ${testResults.duration}s

## Results

${testResults.results.map(r => `
### ${r.name}

- Status: ${r.success ? '✓ PASS' : '✗ FAIL'}
- Duration: ${r.duration}ms
${r.error ? `- Error: ${r.error}` : ''}
`).join('\n')}

---

*Generated at ${new Date().toISOString()}
`;

    return convertMarkdownStringToPDF(markdown, 'test-report.pdf');
}
```

## Troubleshooting

### Issue: Chromium download fails

**Solution:** Use China mirror

```bash
set PUPPETEER_DOWNLOAD_HOST=https://mirrors.huaweicloud.com/chromium-browser-snapshots
npm install mdpdf
```

### Issue: Chinese characters not rendering

**Solution:** Ensure CSS includes Chinese fonts

```css
body {
    font-family: "Microsoft YaHei", "SimHei", "PingFang SC", sans-serif;
}
```

### Issue: PDF generation is slow

**Solution:** First run downloads Chromium (~150-300MB). Subsequent runs are much faster.

### Issue: CSS not applying

**Solution:** Check CSS file path and syntax

```bash
# Use absolute path
node scripts/render.js input.md output.pdf --css /full/path/to/style.css

# Or verify relative path
node scripts/render.js input.md output.pdf --css ./assets/custom.css
```
